### MAC os / Linux basic command

| Command | Description | Example |
| --- | --- | --- |
| ls | List files/directories | ls -l (detailed list) |
| cd | Change directory | cd /home/user |
| pwd | Print working directory | pwd |
| mkdir | Create a directory | mkdir my_folder |
| rm | Remove files/directories | rm file.txt |
| cp | Copy files/directories | cp file.txt /backup/ |
| mv | Move/rename files/directories | mv old.txt new.txt |
| cat | Display file content | cat file.txt |
| grep | Search for text in files | grep "error" log.txt |
| find | Find files/directories | find /home -name "*.txt" |

### Vim editor basic

| Action | Command |
| --- | --- |
| Open file | vim filename |
| Enter insert mode | Press i |
| Exit insert mode | Press Esc |
| Save and quit | :wq + Enter |
| Quit without saving | :q! + Enter |
| Delete a line | dd (in normal mode) |
| Undo | u (in normal mode) |
